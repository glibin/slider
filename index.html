Slider

Simple Javascript slider build with jQuery, CSS3 and no images.

Usage:

Include:

<script src="js/slider.js"></script>
<link href="css/slider/default.css" rel="stylesheet">

HTML:

<div class="slider-container"></div>

Javascript:

jQuery('.slider-container').slider({
    'min' : 0,
    'max' : 100,
    'pos' : 40,
    'change' : function(pos) {
        // Event on change with current pos
    }
});

Available params:

min - minimal value

max - maximal value

pos - default value

width - slider width
